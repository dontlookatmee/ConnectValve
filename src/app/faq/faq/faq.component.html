ConnectValve is a collaboration app. You can create your own services, which will be seen by all users, so they can send you offers. After offer is send by someone, you have choice to accept, decline temporary or delete the offer eternally. If you accept the offer, the user that have send you the offer can create a collaboration, after collaboration is created you and the offer sender have to enter it as soon as possible. If both of you have joined the collaboration you can start the collaboration and a timer will start, after time expires, collaboration will finish and you can send further messages.
